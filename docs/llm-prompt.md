## コンパクト版（桁数判定用）

```
かな語が3桁(100-999)になるか判定してください。

■ 桁数ルール
- 拗音・外来音(きゃ,シュ,ティ等)→ 1単位で2桁 ※最優先で判定
- 1桁かな: れ,お,い,ひ,に,ふ,さ,み,し,よ,こ,ろ,る,な,う,は,や,あ,き,く,か
- 上記以外の単独かな(ま,と,ち,つ,す等)→ 2桁

■ 手順
1. 拗音・外来音をまず1単位にまとめる
2. 各単位の桁数(1 or 2)を合算

例: きれい → き(1)+れ(1)+い(1) = 3桁 ✓
例: にし → に(1)+し(1) = 2桁 ✗
例: きゃく → きゃ(2)+く(1) = 3桁 ✓
例: まち → ま(2)+ち(2) = 4桁 ✗
```

---

## 使い方例

```
上のルールで、以下の単語が3桁になるか判定してください。
- さくら
- ひかり
- はなび
```
