# かな数字対応表

数字をかな（ひらがな・カタカナ）に変換するための対応表。

- 濁音・半濁音は清音と同じ扱い（が=か, ば=は, ぱ=は 等）
- 例外: ジャジュジョ はリャ行と同値（68, 67, 64）
- 「を」は「お」と同じ扱い（= 0）

## 1桁

| 数字 | core | sub | bad |
| ---- | ---- | --- | --- |
| 0    | れ   | お  |     |
| 1    | い   | ひ  |     |
| 2    | に   | ふ  |     |
| 3    | さ   | み  |     |
| 4    | し   | よ  |     |
| 5    | こ   | ら  |     |
| 6    | ろ   | る  |     |
| 7    | な   | う  |     |
| 8    | は   | や  | あ  |
| 9    | き   | く  | か  |
