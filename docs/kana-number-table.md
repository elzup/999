# かな数字対応表

数字をかな（ひらがな・カタカナ）に変換するための対応表。

- 濁音・半濁音は清音と同じ扱い（が=か, ば=は, ぱ=は 等）
- 例外: ジャジュジョ はリャ行と同値（68, 67, 64）
- 「を」は「お」と同じ扱い（= 0）

## スコア計算（3桁辞書向け）

基本数字3桁の辞書へ単語を登録する際、変換品質を評価するスコア。

### トークンスコア（各かな単位）

| 種別            | スコア | 説明                          |
| --------------- | ------ | ----------------------------- |
| core 1桁       | +2     | 高ティア（さ=3, き=9 等）     |
| sub 1桁        | -1     | 低ティア（ら=5, う=7 等）     |
| bad 1桁        | -2     | 最低ティア（か=9, あ=8 等）   |
| 2桁変換        | +1     | 2桁コンパウンド（と=10 等）   |

### 桁数ペナルティ（目標: 3桁）

| 条件       | ペナルティ     |
| ---------- | -------------- |
| はみ出し   | -3 × 超過桁数  |
| 桁不足     | -3 × 不足桁数  |

### 計算例

| 単語     | トークン          | 桁数 | トークン計 | 桁ペナルティ | 合計 |
| -------- | ----------------- | ---- | ---------- | ------------ | ---- |
| きれい   | き(+2) れ(+2) い(+2) | 3    | +6         | 0            | **6**  |
| さとう   | さ(+2) と(+1) う(-1) | 4    | +2         | -3           | **-1** |
| からす   | か(-2) ら(-1) す(+1) | 4    | -2         | -3           | **-5** |
| きゃく   | きゃ(+1) く(-1)      | 3    | 0          | 0            | **0**  |
