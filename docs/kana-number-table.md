# かな数字対応表

数字をかな（ひらがな・カタカナ）に変換するための対応表。

- 濁音・半濁音は清音と同じ扱い（が=か, ば=は, ぱ=は 等）
- 例外: ジャジュジョ はリャ行と同値（68, 67, 64）
- 「を」は「お」と同じ扱い（= 0）

## 桁数判定ルール

かな語が 3桁 (100–999) になるか判定する方法。

1. 拗音・外来音（きゃ, シュ, ティ 等）をまず 1単位にまとめる
2. 各単位の桁数（1桁 or 2桁）を合算

- 1桁かな: れ, お, い, ひ, に, ふ, さ, み, し, よ, こ, ら, ろ, る, え, な, う, は, や, あ, き, く, か, を
- 上記以外の単独かな（ま, と, ち, つ, す 等）→ 2桁
- 拗音・外来音（きゃ, シュ, ティ 等）→ 1単位で2桁

例:

| 単語   | 分解              | 桁数 | 判定 |
| ------ | ----------------- | ---- | ---- |
| きれい | き(1)+れ(1)+い(1) | 3    | OK   |
| にし   | に(1)+し(1)       | 2    | NG   |
| きゃく | きゃ(2)+く(1)     | 3    | OK   |
| まち   | ま(2)+ち(2)       | 4    | NG   |

## スコア計算（3桁辞書向け）

3桁の辞書へ単語を登録する際、変換品質を評価するスコア（targetDigits=3）。

### 計算フロー

```
合計スコア = トークンスコアの合計 + youon4ペナルティ + 先頭0省略ボーナス
```

### トークン分類とスコア

各かなトークンを先頭から順に桁位置（pos）を割り当て、以下の優先順で分類する。

| 優先 | 種別          | スコア | 条件                                           |
| ---- | ------------- | ------ | ---------------------------------------------- |
| 1    | overflow      | -10    | トークン全体が target 外（startPos >= 3）       |
| 2    | sokuon (促音) | +20    | っ/ッ（次のかなの数字をリピート）               |
| 3    | halfOverflow  | +4     | 2桁トークンが境界を跨ぐ（startPos < 3, endPos >= 3） |
| 4    | double (2桁)  | +30    | 2桁変換が target 内に収まる（と, ち, す 等）    |
| 5    | single (core) | +10    | 1桁かな 高ティア（き, さ, い 等）               |
| 5    | single (sub)  | +8     | 1桁かな 中ティア（く, ら, う 等）               |
| 5    | single (bad)  | +6     | 1桁かな 低ティア（か, あ, え 等）               |

### グローバル補正

| 種別             | スコア | 条件                                                |
| ---------------- | ------ | --------------------------------------------------- |
| youon4省略       | -5     | 拗音4字（○ゃ/ゅ/ょ＋う/ん＋○）の3文字目を省略した場合 |
| 先頭0省略ボーナス | +15    | 変換後の桁数が targetDigits 未満（例: 2桁 → 先頭に0を補完） |
| mix              | -7     | 同じ数字が異なるかなで表されている場合（促音は除外） |

### ティア分類

各数字に対する1桁かなのティア。濁音・半濁音は清音と同じティア、カタカナもひらがなに正規化。

| 数字 | core | sub | bad |
| ---- | ---- | --- | --- |
| 0    | れ   | お  |     |
| 1    | い   | ひ  |     |
| 2    | に   | ふ  |     |
| 3    | さ   | み  |     |
| 4    | し   | よ  |     |

| 6    | ろ   | る  | え  |
| 7    | な   | う  |     |
| 8    | は   | や  | あ  |
