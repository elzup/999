# かな数字対応表

数字をかな（ひらがな・カタカナ）に変換するための対応表。

- 濁音・半濁音は清音と同じ扱い（が=か, ば=は, ぱ=は 等）
- 例外: ジャジュジョ はリャ行と同値（68, 67, 64）
- 「を」は「お」と同じ扱い（= 0）

## 桁数判定ルール

かな語が 3桁 (100–999) になるか判定する方法。

1. 拗音・外来音（きゃ, シュ, ティ 等）をまず 1単位にまとめる
2. 各単位の桁数（1桁 or 2桁）を合算

- 1桁かな: れ, お, い, ひ, に, ふ, さ, み, し, よ, こ, ら, ろ, る, え, な, う, は, や, あ, き, く, か, を
- 上記以外の単独かな（ま, と, ち, つ, す 等）→ 2桁
- 拗音・外来音（きゃ, シュ, ティ 等）→ 1単位で2桁

例:

| 単語   | 分解              | 桁数 | 判定 |
| ------ | ----------------- | ---- | ---- |
| きれい | き(1)+れ(1)+い(1) | 3    | OK   |
| にし   | に(1)+し(1)       | 2    | NG   |
| きゃく | きゃ(2)+く(1)     | 3    | OK   |
| まち   | ま(2)+ち(2)       | 4    | NG   |

## スコア計算（3桁辞書向け）

3桁の辞書へ単語を登録する際、変換品質を評価するスコア（targetDigits=3）。

### トークン分類とスコア

| 種別          | スコア | 条件                               |
| ------------- | ------ | ---------------------------------- |
| single (core) | +10    | 1桁かな 高ティア（き, さ, い 等）  |
| single (sub)  | +8     | 1桁かな 中ティア（く, ら, う 等）  |
| single (bad)  | +6     | 1桁かな 低ティア（か, あ, れ, え） |
| double (2桁)  | +30    | 2桁変換（と, ち, す 等）           |
| sokuon (促音) | +20    | っ/ッ（次のかなの数字をリピート）  |
| halfOverflow  | +4     | 2桁が境界を跨ぐ（pos 2–3）         |
| overflow      | -10    | target外（startPos >= 3）          |
| youon4省略    | -5     | 拗音4字の3文字目省略ペナルティ     |

### 計算例

| 単語   | トークン                          | スコア計算 | 合計   |
| ------ | --------------------------------- | ---------- | ------ |
| きれい | き(core+10) れ(bad+6) い(core+10) | 10+6+10    | **26** |
| ちき   | ち(double+30) き(core+10)         | 30+10      | **40** |
| いす   | い(core+10) す(double+30)         | 10+30      | **40** |
| っきき | っ(sokuon+20) き(core+10)         | 20+10      | **30** |
| ちす   | ち(double+30) す(half+4)          | 30+4       | **34** |
